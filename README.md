# SHIVGARJANA-SINNAR-
शिवगर्जना मित्र मंडळ दहीहंडी समिती, सिन्नर ही केवळ दहीहंडी आयोजित करणारी संस्था नाही, तर सामाजिक बांधिलकी जपणारी आणि प्रत्येक गरजू व्यक्तीपर्यंत मदतीचा हात पोहोचवणारी युवा चळवळ आहे. आमचा विश्वास आहे की खरी भक्ती म्हणजे समाजसेवा. म्हणूनच आम्ही वर्षभर विविध सामाजिक, शैक्षणिक, आरोग्यविषयक आणि पर्यावरणपूरक उपक्रम राबवतो.
